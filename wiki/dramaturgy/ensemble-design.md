# Ensemble-Design

Ein Ensemble ist kein Figurenkatalog. Es ist ein Argumentations-Apparat: Jede Figur verkörpert eine Position im Wertegegensatz, und ihre Kollisionen erzeugen den Ideen-Streit der Serie.

---

## 1. Ensemble als Thema-Verkörperung

### Prinzip

Truby: "Each opponent is a variation on the theme; each deals with the same moral problem in a different way." Das Ensemble entsteht nicht aus Casting-Logik, sondern aus dem Wertegegensatz. Jede Figur vertritt eine **Facette** -- eine Position im zentralen Konflikt.

### Ableitungskette

```
WERTEGEGENSATZ (z.B. WÜRDE vs. SICHERHEIT)
       ↓
  4–5 FACETTEN (Positionen im Ideen-Streit)
       ↓
  HUTZLER-ZUORDNUNG (Facette → Enneagramm-Typ)
       ↓
  FIGUREN-SYNTHESE (Typ + Recherche → Mensch)
       ↓
  ENSEMBLE-AUFSTELLUNG (Kreuzungsmatrix + Kollisionspaare)
```

### Regeln

- Die Facette ist ein **Kompass**, keine Eigenschaft. Die Figur LEBT die Position, sie ERKLÄRT sie nicht.
- Thema-Begriffe erscheinen nie im Figurentext. Kein "sie kämpft für Würde" -- zeig, was sie TUT.
- Jede Figur muss als Protagonist ihrer eigenen Geschichte funktionieren (Perspektiv-Test).

## 2. Trubys Opponent-Netz

### Character Web -- Grundprinzip

> "The single biggest mistake writers make when creating characters is that they think of the hero and all other characters as separate individuals." (Truby)

Jede Figur existiert in Relation zu jeder anderen. Vier Verbindungsachsen:

1. **Story-Funktion** (wer treibt was voran?)
2. **Archetyp** (psychologisches Grundmuster)
3. **Thema** (welche Position im moralischen Problem?)
4. **Opposition** (wer greift welche Schwäche an?)

### Rollen im Netz

| Rolle | Funktion | Dramaturgischer Wert |
|---|---|---|
| **Hauptgegner** | Will dasselbe Ziel wie der Held, mit inkompatiblen Mitteln | Definiert den Helden durch Kontrast. Erzwingt wiederkehrenden Konflikt |
| **Sekundärgegner** (2–3) | Greift jeweils eine ANDERE Schwäche des Helden an | Erweitert den Konflikt, verhindert binäre Reduktion |
| **Fake-Ally Opponent** | Scheint Verbündeter, arbeitet gegen den Helden | Stärkste Quelle für Twists und Plotkomplexität. Oft zerrissen von echtem Dilemma |
| **Fake-Opponent Ally** | Scheint Gegner, hilft dem Helden | Weniger nützlich als Fake-Ally Opponent (weniger Konfliktpotential) |
| **Ally** | Helfer des Helden, Resonanzkörper für Werte | Macht innere Zustände des Helden nach außen sichtbar |
| **Subplot-Figur** | Parallel-Track: gleiches Problem, andere Lösung | Kontrastiert den Helden durch Vergleich, nicht durch Konfrontation |

### Four-Corner Opposition

Truby: Mindestens **vier Figuren**, die jeweils eine Ecke besetzen. Jede so verschieden von den anderen wie möglich. Jede greift eine andere Schwäche des Helden an.

```
  HELD ─────────── HAUPTGEGNER
    │                    │
    │    (Werte in       │
    │     Konflikt)      │
    │                    │
  GEGNER 2 ──────── GEGNER 3
```

Vorteile gegenüber Zweier-Opposition:

- Konfliktmenge wächst exponentiell (jede Figur kann mit jeder kollidieren)
- Repräsentiert eine Gesellschaft im Miniaturformat
- Ermöglicht Werte-Cluster statt binärer Moral

**Technik:** Für jede Figur einen Cluster aus verwandten Werten auflisten. Dann positive UND negative Version desselben Werts identifizieren (z.B. entschlossen / aggressiv, ehrlich / unsensibel, patriotisch / herrschsüchtig).

## 3. Kollisionspaare

### Definition

Zwei Figuren, deren Wants oder Lies sich gegenseitig sabotieren und die dadurch in wiederkehrende, eskalierende Konfrontation gezwungen werden.

### Aufbau eines Kollisionspaars

| Element | Frage |
|---|---|
| **Beziehungstyp** | Rivalen, Familie, Verbündete, Liebende? |
| **Oberflächliche Dynamik** | Was sieht man von außen? |
| **Tiefenkonflikt** | Warum können sie nicht voneinander lassen UND nicht zusammen sein? |
| **Want-Sabotage** | Wie verhindert As Want das Need von B? |
| **Gemeinsames Ziel** | Was wollen BEIDE? (Truby: Held und Gegner kämpfen um dasselbe) |
| **Inkompatible Methode** | Warum können sie es nicht GEMEINSAM erreichen? |

### Druck-Beat-Sequenz (McKee)

"Choice under pressure strips away the mask." Drei bis fünf Druckstufen, in denen BEIDE Figuren reagieren:

| Druck | A reagiert | B reagiert | Was der Zuschauer SIEHT |
|---|---|---|---|
| Erste Konfrontation | ... | ... | Masken noch intakt |
| Stärkerer Druck | ... | ... | Risse werden sichtbar |
| Maximaler Druck | ... | ... | Wahre Natur enthüllt |

### Eskalations-Achse

| Episode | Kollision | Was wird schlimmer |
|---|---|---|
| 1–2 | Erste Konfrontation | Noch getarnt als ... |
| 3–4 | Zweite Konfrontation | ... wird sichtbar |
| 5–6 | Dritte Konfrontation | Offener Bruch |
| 7–8 | Vierte Konfrontation | Keine Rückkehr möglich |
| 9–10 | Finale Konfrontation | Endpunkt |

### Mindestanforderung

Drei Kollisionspaare pro Ensemble. Jedes Paar braucht Druck-Beat-Sequenz UND Eskalations-Achse.

## 4. Kreuzungsmatrix

Systematische Erfassung ALLER Figurenbeziehungen. Nicht nur die explosiven Paare, sondern jede Kombination.

### Format

|  | FIGUR A | FIGUR B | FIGUR C | FIGUR D | FIGUR E |
|---|---|---|---|---|---|
| **FIGUR A** | — | Rivalen (wollen denselben Posten) | Fake-Ally (A nutzt C aus) | Unbemerkte Parallele | Asymmetrische Loyalität |
| **FIGUR B** | | — | Subplot-Spiegel | Erotische Spannung | Mentor/Schüler |
| **FIGUR C** | | | — | Klassengegensatz | Alte Schuld |
| **FIGUR D** | | | | — | Geschwister-Dynamik |
| **FIGUR E** | | | | | — |

### Qualitätskriterien

- Jede Zelle enthält die DYNAMIK, nicht nur die Beziehung ("Rivalität um X" statt nur "Rivalen").
- Mindestens drei Zellen enthalten **explosive** Kombinationen (= potentielle Kollisionspaare).
- Keine Figur ist isoliert: Jede hat mindestens zwei substantielle Beziehungen.
- Figuren aus verschiedenen Schichten berühren sich (Brücken-Prinzip).

## 5. POV-Management

### Mittells Attachment-Modell

Mittell (Complex TV): Zuschauer-Bindung entsteht durch **Attachment** (Zeit mit einer Figur verbringen) und **Access** (Zugang zu inneren Zuständen). Beides zusammen bestimmt, mit wem wir mitfühlen.

### Strategien

| Strategie | Beschreibung | Beispiel |
|---|---|---|
| **Centric Episodes** | Eine Episode fokussiert auf eine Figur (Flashback/Backstory) | Lost: "Kate-centric Episode" |
| **Breites Attachment** | Szenen wechseln zwischen vielen Figuren; der ORT wird zum Protagonisten | The Wire: Baltimore als Charakter |
| **Tiers** | Figuren in Rängen: Primary Leads / Secondary Supporting / Tertiary Recurring | Game of Thrones: wechselnde Fokalisierung |
| **Shifting Alignment** | Zuschauer-Sympathie verschiebt sich über die Staffel | Breaking Bad: Walter White |

### Regeln für Serien-Ensembles

1. **Stabiler Kern, bewegliche Peripherie.** Anker-Figuren bleiben, Nebenfiguren rotieren (Mittell: "anchor characters").
2. **Attachment braucht Zeit.** Je mehr Screentime, desto stärker die parasoziale Bindung. Neue Figuren brauchen Etablierung, bevor sie tragende Konflikte übernehmen können.
3. **Wechselnde Fokalisierung vertieft das Thema.** Wenn jede Fokalisierung eine andere Facette des Wertegegensatzes beleuchtet, wird das Ensemble zum prismatischen Argument.
4. **Character-Elaboration statt Character-Change.** Pearson (via Mittell): Serienfiguren akkumulieren Erfahrung und Tiefe, statt sich fundamental zu wandeln. Ausnahmen (Transformation) sind selten und wirkmächtig.

## 6. Ensemble-Größe

### Douglas' Faustregel

- **Pitch/Proposal:** Maximal 3–4 Figuren im Detail. "The listener's eyes will glaze over after you get past your third or fourth character." (Douglas)
- **Fokus-Prinzip:** Eine faszinierende, castable Hauptfigur. Dann Antagonist/Partner mit direkter Verbindung. Restliches Ensemble nur als Tags.
- **In der Serie selbst:** Ensembles in "double digits" funktionieren (The Wire, Game of Thrones), aber erfordern ausgefeiltes POV-Management.

### Praktische Grenzen

| Ensemble-Größe | Geeignet für | Risiko |
|---|---|---|
| 3–4 Figuren | Kammerspiel, Buddy-Formate | Thematische Verengung |
| 5–6 Figuren | Standard-Drama, eine Staffel | Optimales Verhältnis Tiefe/Breite |
| 7–10 Figuren | Ambitionierte Prestige-Serie | Erfordert klare Tier-Struktur |
| 10+ Figuren | Epos (Wire, GoT, Treme) | Narrativer Drive sinkt; braucht strukturelle Gegenmaßnahmen |

### Trubys Warnung vor zu vielen Helden

> "The more characters you must lay out in detail, the more you risk having your story literally come to a halt." (Truby)

**Gegenmaßnahmen** für Multi-Helden-Serien:
- Eine Figur emergiert als zentraler als die anderen
- Alle Figuren teilen dieselbe Desire-Line
- Der Held einer Storyline ist der Opponent einer anderen
- Alle Figuren sind Beispiele eines einzigen Themas
- Cliffhanger am Ende einer Linie triggert den Sprung zur nächsten
- Figuren aus verschiedenen Orten konvergieren an einem Punkt

## 7. Beziehungsdynamiken

### Typen

| Dynamik | Mechanik | Dramaturgischer Ertrag |
|---|---|---|
| **Allianz** | Gemeinsames Ziel, komplementäre Fähigkeiten | Zeigt Werte durch Zusammenarbeit. Wird dramatisch, wenn die Allianz bricht |
| **Rivalität** | Gleiches Ziel, inkompatible Methoden oder Werte | Dauerkonflikt. Stärkstes Mittel zur Figurendefinition durch Kontrast |
| **Asymmetrische Loyalität** | A braucht B, B braucht A nicht (oder umgekehrt) | Machtgefälle erzeugt Spannung und Abhängigkeit |
| **Mentor/Schüler** | Wissensgefälle, das sich über die Staffel schließt | Transformation des Schülers; tragisch, wenn der Mentor überholt wird |
| **Fake-Allianz** | Oberfläche: Verbündete. Tiefe: Gegner | Stärkste Quelle für Twists. Truby: "the most complex and most fascinating characters" |
| **Subplot-Spiegel** | Paralleles Problem, andere Lösung | Kontrastiert den Helden ohne direkte Konfrontation. Verdoppelt das Thema |

### Mittell: Intercharacter-Dynamik statt Character-Change

> "On most series, we watch fairly stable characters interacting to form dynamic relationships, with such interactions providing the surrogate dramatic hook for change." (Mittell)

Die BEZIEHUNGEN verändern sich, nicht die Figuren. Die Figur bleibt im Kern stabil, aber wie andere Figuren sie sehen und behandeln, verschiebt sich. Dramaturgisch entscheidend: nicht ob eine Figur sich wandelt, sondern wie sich das Geflecht um sie herum rekonfiguriert.

## 8. Enneagramm-Interaktionen

### Reibungspotential nach Typ-Kombinationen

Die Hutzler-eBooks definieren pro Typ: Character in an Instant, Fears, Dark Side, Life Lessons, Story Questions. Maximale Reibung entsteht, wenn die **Fears** einer Figur durch die **Dark Side** einer anderen aktiviert werden.

| Typ-Kombination | Reibungsmechanismus | Serienbeispiel |
|---|---|---|
| **1 (Conscience) vs. 8 (Will)** | 1 will moralische Ordnung. 8 ignoriert Regeln. 1s Rigidität provoziert 8s Rebellion, 8s Brutalität bestätigt 1s Angst vor Chaos | Cedric Daniels vs. Jimmy McNulty (The Wire) |
| **3 (Ambition) vs. 4 (Idealism)** | 3 optimiert die Oberfläche. 4 verachtet Oberfläche. 3 entlarvt 4s Selbstmitleid, 4 entlarvt 3s Leere | Don Draper vs. Peggy Olson (Mad Men, partiell) |
| **6 (Truth) vs. 7 (Excitement)** | 6 sichert ab, 7 flieht vorwärts. 6 hält 7 für verantwortungslos, 7 hält 6 für paranoid | Carrie Mathison (6) im Wechselspiel verschiedener Partner |
| **2 (Love) vs. 5 (Reason)** | 2 braucht Nähe und Bestätigung. 5 braucht Distanz und Autonomie. 2s Geben erstickt 5, 5s Rückzug bestätigt 2s Angst, nicht liebenswert zu sein | — |
| **8 (Will) vs. 9 (Imagination)** | 8 dominiert, 9 weicht aus. 8 verachtet Passivität, 9 fürchtet Konfrontation. Explosive Dynamik, wenn 9 endlich rebelliert | Tony Soprano (8) vs. diverse Figuren |
| **1 (Conscience) vs. 3 (Ambition)** | 1 will es richtig machen, 3 will es erfolgreich machen. Gemeinsame Leistungsorientierung, aber inkompatible Definition von "gut" | — |
| **4 (Idealism) vs. 8 (Will)** | 4 leidet expressiv, 8 verachtet Schwäche. 4 sieht 8 als gefühllos, 8 sieht 4 als selbstbezogen | — |

### Integration/Desintegration

Jeder Enneagramm-Typ bewegt sich unter Stress in die Richtung eines anderen Typs (Desintegrationsrichtung) und unter Wachstum in eine andere (Integrationsrichtung). Diese Bewegungen sind dramaturgisch wertvoll:

- **Stress-Richtung** = Dark Side-Verstärkung = Figur unter maximalem Druck zeigt Züge eines anderen Typs
- **Wachstums-Richtung** = Life Lesson-Annäherung = Figur nähert sich ihrem Need

Für Ensemble-Design: Wenn Figur A unter Stress die Züge von Figur Bs Grundtyp annimmt, entsteht eine unerwartete Spiegelung -- die Figuren werden sich ähnlicher, je mehr sie einander bekämpfen.

## 9. Proto-Protagonist / Proto-Antagonist

### Selektion

Aus dem fertigen Ensemble werden zwei Figuren als proto-dramatische Pole identifiziert:

| Rolle | Kriterium |
|---|---|
| **Proto-Protagonist** | Größte Nähe zur **Botschaft** (vorläufige Antwort auf den Wertegegensatz). Nicht der sympathischste, sondern der, dessen Arc die Botschaft verkörpern KANN |
| **Proto-Antagonist** | Stärkste **Gegenposition** zur Botschaft. Nicht der böseste, sondern der, der die überzeugendste Alternative vertritt |

### Regeln

- Proto-Protagonist und Proto-Antagonist müssen um dasselbe Ziel kämpfen (Truby).
- Beide haben moralische Ambivalenz. Der Antagonist ist kein Schurke, sondern vertritt eine Position, die der Zuschauer teilweise nachvollziehen kann.
- Die Selektion ist vorläufig. Im Staffelbogen kann sich herausstellen, dass eine andere Figur stärkeren narrativen Drive hat.
- Der Proto-Protagonist ist der, dessen Want-Need-Konflikt den Zuschauer am stärksten packt, nicht der, der am meisten Recht hat.

## 10. Von Facetten zum funktionierenden Ensemble -- Praktischer Workflow

### Schritt 1: Facetten aus dem Thema (Phase 3)

4–5 Facetten, die verschiedene Positionen im Wertegegensatz verkörpern. Jede Facette ist eine RICHTUNG, keine Eigenschaft.

### Schritt 2: Hutzler-Zuordnung (Phase 4)

Jede Facette wird einem Enneagramm-Primär-Typ zugeordnet (via Enneagram-Werte-Matrix). Sekundär-Typ für zusätzliche Färbung.

### Schritt 3: Figuren-Synthese (Phase 5, Agent-Kette)

Pro Figur, sequenziell:
1. **Ghost + Lie** (aus Hutzler Character in an Instant + Fears + Recherche)
2. **Want + Need + Zerreißprobe** (aus Ghost/Lie-Derivation + Hutzler Life Lessons)
3. **Fears + Dark Side + Paradox** (aus Hutzler Fears/Dark Side)
4. **Hier und Jetzt + Konsolidierung** (Manifestation im konkreten Milieu)

### Schritt 4: Kreuzungsmatrix erstellen

Jede Figurenkombination prüfen: Was ist die Dynamik? Wo entsteht Reibung? Tabelle ausfüllen.

### Schritt 5: Kollisionspaare identifizieren

Mindestens drei Paare aus der Matrix, die wiederkehrende Reibung erzeugen. Für jedes Paar: Gemeinsames Ziel, inkompatible Methode, Druck-Beat-Sequenz, Eskalations-Achse.

### Schritt 6: Proto-Protagonist / Proto-Antagonist selektieren

Wer ist der Botschaft am nächsten? Wer ist die überzeugendste Gegenposition?

### Schritt 7: Dangling Causes pflanzen

Drei Details, die früh etabliert werden und später explodieren. Pro Cause: Pflanzung (Episode) → Schlaf (Episoden) → Zündung (Moment) → Detonation (Konsequenz).

### Validierungs-Checkliste

- [ ] Jede Figur vertritt eine Position im Wertegegensatz
- [ ] Kreuzungsmatrix vollständig, keine isolierte Figur
- [ ] Mindestens 3 Kollisionspaare mit Druck-Beat-Sequenz und Eskalations-Achse
- [ ] Proto-Protagonist und Proto-Antagonist identifiziert und begründet
- [ ] Trubys Four-Corner Opposition erkennbar (mindestens 4 Figuren in wechselseitiger Opposition)
- [ ] Jede Figur funktioniert als Protagonistin ihrer eigenen Geschichte
- [ ] POV-Strategie skizziert: Wessen Geschichte erzählen wir wann?
- [ ] 3 Dangling Causes mit Pflanzung-Schlaf-Zündung-Detonation
- [ ] Gesellschaftsspektrum abgedeckt: Figuren aus mindestens zwei verschiedenen Schichten
