# Character Rules — Binding Reference for All Character Agents

All character agents (`ghost-lie`, `want-need`, `abyss`, `manifestation`) must comply with these rules. No exceptions.

---

## 1. The 11 Dramatic Units

| # | Unit | Core Question | Agent |
|---|------|---------------|-------|
| 1 | **Ghost** | What formative past event made the character who they are? | ghost-lie |
| 2 | **Lie** | What false belief did the character derive from the Ghost? | ghost-lie |
| 3 | **Here and Now** | How does the character live at the start — what do they DO? | manifestation |
| 4 | **Want** | What does the character pursue — concrete, physical, graspable? | want-need |
| 5 | **Need** | What does the character actually lack — unconscious, deeper? | want-need |
| 6 | **Crucible** | Where do Want and Need collide irreconcilably? | want-need |
| 7 | **Fears** | Core Fear + Secret Shame. The fear beneath the fear. | abyss |
| 8 | **Dark Side** | Behavior under maximum pressure — concrete actions when all strategies fail. | abyss |
| 9 | **Arc** | Inner transformation (or failure to transform). | structure phase |
| 10 | **Turning Points** | Concrete moments that force direction changes in the Arc. | structure phase |
| 11 | **Paradox** | ONE trait that is simultaneously strength AND weakness. | abyss |

### Connections Between Units

```
Ghost ──► Lie ──► Want ──► Crucible ◄── Need
  │                                       ▲
  └──► Fears ──► Dark Side               │
                                    Life Lessons
```

- Ghost CAUSES Lie (causal, psychological).
- Lie DRIVES Want (the false belief produces the external goal).
- Want OPPOSES Need (pursuing Want prevents fulfilling Need).
- Crucible is the MECHANISM of opposition (not a moment — a structural trap).
- Fears are the SCAR of the Ghost (its long-term effect).
- Dark Side is where Fears lead when unchecked.
- Paradox threads through Ghost, Lie, Want, AND Dark Side — one trait visible in all four.

## 2. Derivation Chain — Forward Only

```
Enneagram Type (Character in an Instant + Fears)
    → GHOST (formative event, synthesized with research material)
    → LIE (false belief derived from Ghost)
    → WANT (conscious goal driven by Lie)
    → NEED (unconscious deficit — opposite of Fatal Flaw)
    → CRUCIBLE (Want PREVENTS Need — mechanical opposition)
```

### Rules

| Rule | Meaning |
|------|---------|
| **Forward derivation only** | Ghost → Lie → Want → Need → Crucible. Never backwards. Never "character needs X, so they have Y." |
| **Ghost determines Lie** | The Lie must be a logical (but fatal) conclusion FROM the Ghost. Not a moral judgment by the author. |
| **Want from Lie** | The Want is what the Lie tells the character to pursue. |
| **Need from Life Lessons** | The Need corresponds to the Enneagram type's Life Lessons. It is the OPPOSITE of the Fatal Flaw. |
| **Crucible is mechanical** | Pursuing the Want must STRUCTURALLY worsen the Need deficit. Not "it's hard" — it's a trap. |

### What Must Exist Before What

| To create... | You need... |
|---|---|
| Lie | Ghost |
| Want | Ghost + Lie |
| Need | Ghost + Lie + Facet position |
| Crucible | Want + Need |
| Fears | Ghost + Enneagram type |
| Dark Side | Fears + Lie |
| Paradox | Ghost + Lie + Want + Dark Side |
| Here and Now | All previous units |
| Event Building Blocks | All previous units + ensemble context |

## 3. Concreteness Mandate

From the character phase onward: **No sentence that could appear in an essay.**

### Per Unit

| Unit | MUST be | MUST NOT be |
|------|---------|-------------|
| **Ghost** | A specific scene — place, people, actions, sensory details | A category of experience ("grew up in poverty") |
| **Lie** | A behavioral pattern — observable, filmable | An internal belief stated abstractly ("trusts no one") |
| **Want** | An active goal — something the character DOES | A feeling or state ("wants to be happy") |
| **Need** | An unconscious deficit — the character doesn't know it | A conscious wish |
| **Crucible** | A mechanical opposition — structural, demonstrable | A vague tension |
| **Fears** | Core Fear as identity terror + Secret Shame as hidden conviction | External consequences ("fears losing the job") |
| **Dark Side** | Concrete actions under pressure — what they specifically DO | Adjectives ("becomes cruel") |
| **Paradox** | ONE trait — named, with examples of strength and weakness | Two traits in opposition |
| **Here and Now** | Playable — an actor could perform it | Psychological description |

### The Essay Test

> Could this sentence appear in an essay about the Value Conflict?
> If yes → rewrite into concrete action.

**FAIL:** "Lena became an expert in stability because stability was a riddle in her home."
**PASS:** "Lena's mother disappeared three times a year. Lena was eight. She kept a calendar marking her mother's mood in colors."

## 4. Show, Don't Tell

### Five Channels of Emotion (Puglisi)

| Channel | Controllability | Honesty |
|---------|----------------|---------|
| **Dialogue** | High — consciously steerable | Lowest — lies most |
| **Vocal Cues** | Medium | Medium |
| **Body Language** | Medium | Medium |
| **Thoughts** | Variable (POV only) | Variable |
| **Visceral Reactions** | None — purely instinctive | Highest — never lies |

**Rule:** The less controllable the channel, the more honest. Use visceral reactions and body language to SHOW what dialogue conceals.

### Prohibitions

| Prohibited | Replacement |
|---|---|
| Author commentary ("What X doesn't see:") | Stay inside the character's experience |
| Meta-explanation ("She represents...") | Show through action |
| Essay language ("Order is her access to the world") | Concrete sensory detail |
| Motivational slogans ("Those who submit get crushed") | Specific behavior in a specific situation |
| Psychological explanation ("because stopping would mean...") | The action itself |
| Negation-explanation ("Not anger, not sadness, but...") | State what IS |

## 5. Anti-Sanitization Rules for Characters

| # | Rule | Enforcement |
|---|------|-------------|
| AS1 | **No moralizing** — no enlightening speeches unless built into the arc | FAIL if violated |
| AS2 | **No spontaneous redemption** — toxic characters STAY toxic if the arc dictates. Does the character have a redemption arc in the sheet? If not — no insight, no remorse. | FAIL if violated |
| AS3 | **No balancing justice** — not every act gets punished | FAIL if violated |
| AS4 | **No disclaimers** — no "This is of course problematic" | FAIL if violated |
| AS5 | **No linguistic softening** — no euphemisms for a character's language or actions | FAIL if violated |
| AS6 | **No victim empowerment as default** — some characters are destroyed | FAIL if violated |
| AS7 | **No symmetrical portrayal** — no "positive counterexample" in the same scene | FAIL if violated |
| AS8 | **No tonal dilution** — hope is earned, never given | FAIL if violated |

## 6. Style Rules for Character Text

### Anti-AI Patterns (Prohibited)

| Pattern | Example | Threshold |
|---------|---------|-----------|
| Negation-explanation | "Not anger, not sadness, but..." | > 3x = FAIL |
| Pseudo-depth | "Can't distinguish speed from flight" | > 3x = FAIL |
| Psychological explanation | "because stopping would be the same as..." | > 3x = FAIL |
| Comma before "and" | ", and" | > 5x = FAIL |
| Sentences > 35 words | — | Flag each |

| # | Criterion | Consequence |

## 7. PASS vs. FAIL per Unit

### Ghost — PASS if:
- [ ] Specific event — place, time, people, sensory detail
- [ ] Psychologically dense — this Ghost leads to THIS Lie and no other (or the layers are shown)
- [ ] Not exposition-dumped — revealed layer by layer
- [ ] Derived from Enneagram primary type (Character in an Instant + Fears)

### Ghost — FAIL if:
- Category instead of event ("grew up poor")
- Could lead to any Lie equally well (too generic)
- Author explains the Ghost's significance instead of showing it

### Lie — PASS if:
- [ ] Behavioral and observable — you can SEE it in how the character acts
- [ ] Formulable as a sentence the character believes ("The world works like this: ...")
- [ ] Derived from Ghost — causal chain intact
- [ ] Matches Enneagram secondary type's World View

### Lie — FAIL if:
- Abstract psychological state ("has trust issues")
- Not derivable from the Ghost
- Author's moral judgment instead of character's coping strategy

### Want — PASS if:
- [ ] Active — something the character DOES, not feels
- [ ] Concrete and physical — could be filmed
- [ ] Ego-driven — directly benefits the character
- [ ] Driven by the Lie

### Want — FAIL if:
- A feeling or state ("wants to be at peace")
- Not specific enough to drive plot
- Compatible with the Need (no story if both can be achieved)

### Need — PASS if:
- [ ] Unconscious — the character doesn't know it, denies, suppresses, or ignores it
- [ ] Opposite of the Fatal Flaw
- [ ] Corresponds to Enneagram type's Life Lessons
- [ ] In genuine conflict with Want

### Need — FAIL if:
- The character is aware of it
- Compatible with Want
- Same as Want but phrased differently

### Crucible — PASS if:
- [ ] Mechanical — pursuing Want STRUCTURALLY prevents fulfilling Need
- [ ] Demonstrable — you can trace the mechanism step by step
- [ ] Escalating — gets worse the harder the character pursues Want

### Crucible — FAIL if:
- Vague tension ("it's hard to balance both")
- Only a single moment (it's a structural trap, not an event)
- Want and Need could coexist

### Fears — PASS if:
- [ ] Core Fear is identity-level ("who I AM"), not external consequence
- [ ] Secret Shame is a conviction about themselves they must hide
- [ ] Derived from Ghost and Enneagram type
- [ ] Present as undercurrent in EVERY scene (drives, blocks, distorts, or escalates)

### Fears — FAIL if:
- External consequence only ("fears losing the job")
- Not connected to Ghost
- Only activated in crisis moments (must be continuous)

### Dark Side — PASS if:
- [ ] Concrete actions — what they specifically DO under maximum pressure
- [ ] Logical consequence of unchecked Fears + Lie pushed to extreme
- [ ] 5-step Dark Side Ladder with escalating pressure events
- [ ] Answers: "Where is this character in 5 years if nothing changes?" with "Death" (physical or metaphorical)

### Dark Side — FAIL if:
- Adjectives instead of actions ("becomes cruel")
- Not derivable from Fears
- Too mild — 5-year prognosis isn't death

### Paradox — PASS if:
- [ ] ONE trait — named explicitly
- [ ] Demonstrably strength AND weakness with concrete examples
- [ ] Visible in Ghost, Lie, Want, AND Dark Side
- [ ] Generates friction that could carry 10 seasons

### Paradox — FAIL if:
- Two traits in opposition instead of one trait that is both
- Not traceable through the other units
- Resolvable (a true Paradox never fully resolves)

### Here and Now — PASS if:
- [ ] Playable — an actor could perform it without further instruction
- [ ] Symptoms of the Lie visible in everyday behavior
- [ ] Emotional baseline defined (demonstrative vs. reserved, fuse length)
- [ ] Details NOT derivable from the theme (surprising but fitting)

### Here and Now — FAIL if:
- Psychological description instead of observable behavior
- All details are theme-derived (no individuality)
- An actor couldn't play it

## 8. Ensemble-Level Character Rules

### Ensemble Depth by Tier

| Tier | Required Units | Depth |
|------|----------------|-------|
| **Proto-Protagonist** | All 11 | Maximum |
| **Main Ensemble (4-6)** | Ghost, Lie, Want, Need, Fears, Arc | High |
| **Supporting Ensemble** | Want, Fears, Paradox | Functional |
| **Episode Characters** | Want + one dominant trait | Minimal |

### Individuality Test

> Could ANOTHER character with the same Enneagram type be described the same way?
> If yes → not individual enough.

### Facet as Compass

The facet from the theme gives DIRECTION, not a trait. The character LIVES the position — never EXPLAINS it. Theme words never appear in character text.
