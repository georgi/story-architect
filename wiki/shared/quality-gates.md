# Quality Gates — Mechanical Pass/Fail Checklists per Phase

Binary checks. Yes/No per point. No interpretation. 100% required from iteration 3 onward.

Gate Score = YES count / Total points x 100%.

---

## Gate 0: Universal (EVERY Phase)

- [ ] Glossary compliance (only terms from `process/00-glossary.md`)
- [ ] No self-censorship
- [ ] No disclaimers
- [ ] No softening (HBO standard)
- [ ] Dark Side shown through concrete actions
- [ ] Author's voice perceptible
- [ ] No author commentary (no "What X doesn't see:")

---

## Gate 1: Theme & Value Conflict

- [ ] Traceable to impulse + research
- [ ] BOTH values pass ambivalence test (saving AND destructive)
- [ ] Machine test passed (Criteria 1-4 + Anti-Value check + Prohibition test)
- [ ] Grounding obligation: min. 2 destructive arguments per value with research material
- [ ] 4-5 facets (max 5), independent positions
- [ ] Tone obligation: core arguments SHARP, not academic
- [ ] Facet dispute: three-stage ambivalence per collision
- [ ] Provisional message formulated
- [ ] NO characters — argumentative level only

---

## Gate 2a: Enneagram Type Assignment

- [ ] Every facet: PRIMARY and SECONDARY type
- [ ] Primary type: Ghost foundation, Fears, Dark Side, Life Lessons
- [ ] Secondary type: World View, Story Questions, relationship patterns
- [ ] Matrix lookup documented
- [ ] No primary type duplicates in ensemble
- [ ] Triad distribution checked (Heart/Head/Body)
- [ ] Conflict potential between types identified

---

## Gate 2b: Character Synthesis

### Per Character — Enneagram Foundation

- [ ] Ghost synthesized from Character in an Instant + Fears of primary type
- [ ] World View from Enneagram type derived into Lie
- [ ] Life Lessons recognized as Need potential
- [ ] Dark Side formulated in concrete actions

### Per Character — Dramatic Units

- [ ] All units filled: Ghost, Lie, Here and Now, Want, Need, Crucible, Fears, Dark Side, Paradox
- [ ] Derivation intact: Ghost → Lie → Want (no backward derivation)
- [ ] Crucible: Want PREVENTS Need ACTIVELY
- [ ] Paradox: ONE trait = Strength AND Weakness
- [ ] Value Conflict tension named (character carries BOTH sides)
- [ ] No Arc decisions yet — only the raw character

### Per Character — Event Building Blocks

- [ ] GAP sequence (McKee): 4 attempts with escalating risk
- [ ] CATCH test (Chamberlain): Point of No Return with wanted + unwanted
- [ ] Fatal Flaw prognosis (Marks): Origin, exhaustion, clinging reflex, prognosis
- [ ] Self-Revelation sketch (Truby): Psychological + Moral + Proof through action
- [ ] Dark Side Ladder (Hutzler + McKee): 5 steps with concrete pressure events

### Per Character — Ambivalence Tests

- [ ] Strength = Weakness?
- [ ] Perspective test: Functions as protagonist of their own story?
- [ ] Dilemma check: Choice without perfect solution?
- [ ] Empathy test: Audience understands the WHY?

### Per Character — Style

- [ ] Ghost is a specific scene, not a category
- [ ] Lie is behavioral and observable
- [ ] Want is active and concrete (could be filmed)
- [ ] Need is unconscious (character doesn't know it)
- [ ] Dark Side is concrete actions, not adjectives
- [ ] Here and Now is playable (an actor could do it)
- [ ] Concreteness mandate met: no essay sentences
- [ ] All units trace to facet position AND Enneagram type

---

## Gate 2c: Ensemble

- [ ] Cross-reference matrix complete (every character vs. every character)
- [ ] Min. 3 collision pairs with: shared goal + incompatible method + pressure-beat sequence (3-5 beats) + escalation axis
- [ ] Collision pairs create RECURRING friction, not one-time conflicts
- [ ] Proto-Protagonist and Proto-Antagonist identified and justified
- [ ] Truby's Four-Corner Opposition recognizable (min. 4 characters in mutual opposition)
- [ ] Every character functions as protagonist of their own story
- [ ] POV strategy sketched: whose story when?
- [ ] 3 Dangling Causes with planting → sleep → ignition → detonation
- [ ] Social spectrum covered: characters from at least two different strata
- [ ] No primary type duplicates
- [ ] Triad diversity (Heart/Head/Body)
- [ ] Hero of one storyline = Opponent of another (Scene Weave)

---

## Gate 3: World

- [ ] 4-8 contrasting sub-worlds derived from character Needs
- [ ] World rules: What is IMPOSSIBLE?
- [ ] Borders and power gradients between sub-worlds
- [ ] Every sub-world thematically linked to Value Conflict
- [ ] Every character: main sub-world with friction test + Ghost echo
- [ ] Min. 3 friction tests for border crossings
- [ ] Min. 3 reusable collision locations
- [ ] Conflict world identified (most dramatic overlap)
- [ ] No decoration — every sub-world as dramatic engine
- [ ] Friction test sentence filled: "Every time [CHARACTER] enters [SUB-WORLD], [WHAT] is activated — because [GHOST/NEED LINK]"

---

## Gate 4: Tone

- [ ] Synthesized from 3 sources (Value Conflict → base tone, Ensemble → dialogue style, Conflict world → atmosphere)
- [ ] Genre named uncompromisingly
- [ ] Atmosphere visual AND acoustic (sensory, not generic)
- [ ] Reference matrix precise (X meets Y, without Z)
- [ ] Pitch tone (1 sentence)
- [ ] Ruthlessness check: Does the tone transport the ugliest truth?

---

## Gate 5: Format

- [ ] Content determines form
- [ ] Runtime + episode count justified
- [ ] Serialization degree clearly defined
- [ ] Platform target consistent with tone

---

## Gate 6: Structure (TV Season Arc)

### 6a Arc Types
- [ ] Every character: Arc type (A/B/C/D) + outcome
- [ ] Mix of arc types in ensemble

### 6b Movement Logic
- [ ] Want runs counter to Need
- [ ] Midpoint = greatest external success + greatest internal loss
- [ ] Act 3: Active choice between Want and Need

### 6c Relationship Dynamics
- [ ] 3 relationship phases per collision pair
- [ ] Min. 1 new crossing through arcs

### 6d Causal Chain
- [ ] >= 60% character-driven
- [ ] No gaps
- [ ] Every Want attempt has Need costs
- [ ] Rising stakes
- [ ] Min. 1 secret with Dramatic Irony

### 6e Sequences
- [ ] 8 sequences each with Want attempt, obstacle, resolution
- [ ] Main tension resolved end of Act 2 (not season finale)
- [ ] Pilot: Pilot launch + Series launch
- [ ] Every episode ending = crossroads

### 6f Beat Structure
- [ ] Every beat: Exposition → Confrontation → Resolution
- [ ] Reversal principle: Act 2 shows OPPOSITE of Act 3
- [ ] Beat protagonist defined
- [ ] Beat motivation ≠ Series Want
- [ ] Action chain: Decision in Act 3 triggers next beat

### 6g Scene Weave
- [ ] Thematic unity: All storylines = variations of the same Value Conflict
- [ ] Juxtaposition: Scenes comment on each other
- [ ] Hero = Opponent of another storyline
- [ ] Funnel: All storylines lead to the same climax
- [ ] Episode question derived from the Idea Battle

### 6h Validation
- [ ] Min. 2 reference constellations per collision pair
- [ ] Every facet represented in min. 1 sequence
- [ ] Climax shows message through ACTION (not dialogue)

---

## Gate 7: Pitch & Export

- [ ] Logline <= 50 words (Protagonist + Want + Crucible)
- [ ] Tagline <= 10 words
- [ ] Pitch structure: META → Opening → World → Characters → Plot teaser
- [ ] Characters and world in foreground, not plot
- [ ] Theme perceptible, not named
- [ ] Author's voice recognizable

|-----------|---------|--------|

| Substance 3.0-3.9 | Functional, generic | TARGETED SHARPENING |

## Backjump Principle

On structural failure, return to the ERROR SOURCE, not the last phase. Then forward.

| Error in | Backjump to |
|----------|-------------|
| Theme (Gate 1) | Research |
| Type Assignment (Gate 2a) | Theme |
| Characters (Gate 2b) | Type Assignment |
| World (Gate 3) | Characters |
| Tone (Gate 4) | World |
| Format (Gate 5) | Tone |
| Season Arc (Gate 6) | Structure sub-phase |
| Pitch (Gate 7) | Season Arc |
