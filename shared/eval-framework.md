# Evaluation Framework

Three evaluation instruments, applied in sequence. Every deliverable must pass all three.

---

## Pre-Check: Anti-Sanitization (mandatory)

Before any evaluation, run the 8-rule anti-sanitization check. If ANY rule is violated, the deliverable is rejected. No scoring. Fix first.

| # | Rule | Pass? |
|---|---|---|
| AS1 | No moralization — no enlightening speeches unless arc-mandated | |
| AS2 | No spontaneous redemption — toxic characters STAY toxic unless arc specifies change | |
| AS3 | No balancing justice — not every act is punished | |
| AS4 | No disclaimers — no "this is of course problematic" | |
| AS5 | No linguistic defusing — no euphemisms for a character's language | |
| AS6 | No default victim empowerment — some characters are destroyed | |
| AS7 | No symmetric portrayal — no "positive counterexample" in the same scene | |
| AS8 | No tonal dilution — hope is earned, never given | |

---

## Instrument 1: Quality Gate (Mechanical Pass/Fail)

See `shared/quality-gates.md` for the full checklist.

Binary. Each gate item is YES or NO. No partial credit. If a gate fails, the deliverable goes back to the error source (not the last phase).

**Purpose:** Catch structural and procedural errors before evaluating quality.

---

## Instrument 2: Dramatic Substance (D1-D7)

Seven dimensions, each scored on a 5-point scale.

| Code | Dimension | What It Measures |
|---|---|---|
| D1 | Specificity | Concrete details, unique to THIS project, not interchangeable |
| D2 | Dramatic Friction | Tension between characters, within characters, between desire and reality |
| D3 | Ambivalence | Moral complexity — viewer can see multiple valid perspectives |
| D4 | Derivation Fidelity | How precisely the output follows from its inputs (theme->character, character->arc, etc.) |
| D5 | Author's Voice | Match with the established voice profile — rhythm, imagery, vocabulary, worldview |
| D6 | Ruthlessness | Willingness to follow dramatic logic to uncomfortable places |
| D7 | Consistency | Internal coherence across all project entities |

### Scoring Guidelines

| Score | Meaning | Calibration |
|---|---|---|
| **5** | Exceptional | HBO/A24 pitch deck quality. Specific, surprising, inevitable. Could not belong to any other project. |
| **4** | Strong | Clear dramatic substance with minor room for improvement. Distinctive but not yet irreplaceable. |
| **3** | Adequate | Correct but interchangeable. Could be transplanted to a different project with minimal adjustment. This is the danger zone — competent mediocrity. |
| **2** | Weak | Generic, predictable, or derivative. Dramatic logic present but not felt. |
| **1** | Failing | Fundamentally flawed. Wrong derivation, missing logic, sanitized, or incoherent. |

**Target:** Every dimension at 4+. A score of 3 on any dimension triggers revision. A score of 1-2 on any dimension triggers return to error source.

---

## Instrument 3: Gold Standard Comparison (V1-V4)

Compare the deliverable against concrete reference works (identified during tone/format phase).

| Code | Dimension | What It Measures |
|---|---|---|
| V1 | Structural Fidelity | Does the architecture match the ambition level of the reference? |
| V2 | Detail Depth | Is the specificity of characterization, world, and scene comparable? |
| V3 | Tonal Fit | Does the deliverable live in the same tonal register it claims? |
| V4 | Originality | Despite structural similarity, is the voice and vision distinct? |

### Scoring Guidelines

Same 5-point scale as Dramatic Substance.

**V4 (Originality) is the most important.** A project that scores 5 on V1-V3 but 2 on V4 is a well-executed copy. A project that scores 3 on V1-V3 but 5 on V4 has a future — it just needs structural work.
