# Quality Gates

Mechanical checklist. YES/NO per point. No interpretation. No "partially." It passes or it doesn't.

---

## Fallback Principle

When a gate fails, go back to the **ERROR SOURCE**, not the last phase.

If the season arc fails because character motivation is unclear — fix the CHARACTER, not the arc. If the pitch fails because the theme is vague — fix the THEME, not the pitch.

The derivation chain is a one-way street. Errors propagate forward. Fixes propagate backward.
