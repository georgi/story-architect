# 07 FORMAT — Structural Frame

## Purpose

Define the structural container for the narrative. Content determines form — never the reverse. Every structural decision is justified by what the material demands, not by convention or market logic.

---

## Prerequisites

- `wiki/projects/[PROJECT]/tone.md` — Pacing requirements
- `wiki/projects/[PROJECT]/characters/*.md` — Ensemble size and complexity
- `wiki/projects/[PROJECT]/world.md` — Conflict World scope

---

## Core Principle

**Content determines form.** The ensemble size, the pacing demands, and the world scope dictate the format. Not: "We want a 10-episode limited series." But: "This ensemble of 6 characters with 4 collision pairs across 5 sub-worlds requires approximately 8-10 episodes of 50 minutes to develop fully."

---

## Three Sources Evaluated

### From Tonality → Pacing Demands
- Does the pacing require long breathing room or rapid compression?
- How much screen/page time does each scene type need?
- What rhythm does the dialogue style demand?

### From Ensemble → Character Count Demands
- How many characters need significant arc development?
- How many Collision Pairs need screen time?
- Can all arcs resolve in a single contained story, or do they require serial development?

### From Conflict World → Scope Demands
- How large is the world that needs to be established?
- How many sub-worlds need to be visited?
- Does the Conflict World require buildup or can it be reached quickly?

---

## Format Parameters by Medium

### TV Series

| Parameter | Options | Determined By |
|-----------|---------|---------------|
| **Episode Count** | 6 / 8 / 10 / 12+ | Ensemble size + arc complexity |
| **Runtime per Episode** | 30 / 45 / 50 / 60 min | Pacing demands + storyline count |
| **Serialization Degree** | Horizontal (continuous) / Vertical (episodic) / Hybrid | Story structure + ensemble interdependence |
| **Season Rhythm** | Cold open? Recurring structure? Episode arcs? | Tone + genre conventions |
| **Platform Target** | Streaming / Cable / Network / Public | Tone + content + audience |

**Serialization Degree** (critical for TV):
- **Horizontal:** Continuous storylines across all episodes. Requires commitment. Rewards binge.
- **Vertical:** Each episode self-contained. Accessible. Limits deep arc development.
- **Hybrid:** A-story serialized, B/C-stories episodic. Most common for character-driven drama.

### Film

| Parameter | Options | Determined By |
|-----------|---------|---------------|
| **Total Runtime** | 90 / 105 / 120 / 150 min | Story scope + pacing |
| **Act Structure** | 3-act / 4-act / non-linear | Arc type + tone |
| **Sequence Count** | 6 / 8 / 10 | Complexity + rhythm |

### Game

| Parameter | Options | Determined By |
|-----------|---------|---------------|
| **Chapter/Act Count** | 3-5 / 5-8 / 8+ | Narrative scope |
| **Estimated Playtime** | 5-10h / 10-20h / 20-40h / 40h+ | Content density |
| **Linearity Degree** | Linear / Branching / Open World | Player agency requirement |
| **Platform** | PC / Console / Mobile / VR | Interaction model |

---

## Format Justification

For EACH parameter chosen, provide a one-sentence justification from content. Not "industry standard" or "audience preference" — but WHY this material needs THIS frame.

**Example:**
- "10 episodes because the 6 character arcs each need a minimum of 2 episodes of focused development, plus shared climax episodes."
- "50-minute runtime because the dialogue-heavy pacing requires breathing room that 30 minutes cannot provide."
- "Hybrid serialization because the A-story (protagonist's Want pursuit) is continuous, but each episode's B-story can spotlight a different Collision Pair."

---

## Quality Gate

Before completing format, verify:

- [ ] Content determines form — every parameter justified from material, not from convention
- [ ] All parameters defined for the chosen medium
- [ ] Serialization degree defined and justified (TV)
- [ ] Platform target consistent with tone and content
- [ ] Ensemble can be adequately served in the chosen format (no character gets shortchanged)
- [ ] Pacing demands met by runtime and episode structure
- [ ] World scope fits the format (not cramming an epic into 90 minutes, not stretching a chamber play to 10 episodes)
