# 03 PSYCHOLOGY MAPPING — Enneagram Type Assignment

## Purpose

Assign each facet an Enneagram type that provides deep psychological architecture: Ghost patterns, Fears, Dark Side behaviors, and Life Lessons. This bridges the abstract Value Conflict to concrete character psychology.

---

## Prerequisites

- `wiki/projects/[PROJECT]/theme.md` — Value Conflict and facets must exist
- `wiki/dramaturgy/enneagram-types.md` — Values Matrix section

---

## Steps

### Step 1: Read Theme

Read `wiki/projects/[PROJECT]/theme.md`. For each facet, understand:
- What position does this facet take in the Value Conflict?
- What value does this facet prioritize?
- What does this facet fear about the opposing value?

### Step 2: Read Enneagram Values Matrix

Read `wiki/dramaturgy/enneagram-types.md`, specifically the Values Matrix section. Understand what each type's core values, fears, and behavioral patterns are.

### Step 3: Assign Types Per Facet

For each facet:

#### A. Identify PRIMARY Type
The primary type provides the **core psychology** — the deep structure:
- **Ghost pattern** — What kind of formative experience does this type produce?
- **Fears** — What is the core fear? What is the secret shame?
- **Dark Side** — How does this type behave under maximum pressure?
- **Life Lessons** — What would this type need to learn?

Match: Which type's **primary values** most naturally align with the facet's position in the Value Conflict?

#### B. Identify SECONDARY Type
The secondary type provides the **behavioral style** — the surface presentation:
- **World View** — How does this type see and interact with the world?
- **Story Questions** — What dramatic questions does this type generate?
- **Relationship patterns** — How does this type connect with and push away others?

Match: Which type's **behavioral patterns** best serve the facet's active behavior?

#### C. Document the Assignment
For each facet, record:
- Primary type + reasoning (why this type's core values match the facet)
- Secondary type + reasoning (why this type's behavior serves the facet)
- Key psychological traits drawn from both types
- Predicted conflict potential with other ensemble types

### Step 4: Validate Ensemble

#### No Primary Type Duplicates
Every character in the ensemble must have a unique primary type. Two characters with the same core psychology will feel redundant.

If duplicates emerge → re-examine which facet can be served by a different but equally valid type.

#### Triad Distribution Check
The Enneagram has three triads:
- **Heart Triad (2, 3, 4)** — Identity, image, shame
- **Head Triad (5, 6, 7)** — Security, anxiety, planning
- **Gut Triad (8, 9, 1)** — Autonomy, anger, control

Check: Is at least one type from each triad represented? A pure Heart-triad ensemble lacks grounding. A pure Gut-triad ensemble lacks vulnerability.

Imbalance is acceptable if dramaturgically justified — but must be conscious.

#### Conflict Potential Mapping
Identify which type pairings create maximum friction:
- Which types have opposing core fears?
- Which types trigger each other's Dark Side?
- Which types cannot understand each other's World View?

These pairings predict your strongest Collision Pairs.

---

## Quality Gate

Before completing psychology mapping, verify:

- [ ] Every facet has a primary AND secondary Enneagram type
- [ ] All assignments traced back to Values Matrix (not intuition)
- [ ] No primary type duplicates in ensemble
- [ ] Triad distribution checked (imbalances justified if present)
- [ ] Conflict potential between types identified
- [ ] Assignments strengthen — not contradict — facet positions in Value Conflict

---

## Output

Adds a **"Psychology Mapping"** section to `wiki/projects/[PROJECT]/theme.md`.

Per facet:
- Facet name + position statement (from theme)
- Primary type: [number] — [reasoning]
- Secondary type: [number] — [reasoning]
- Key traits: [list drawn from both types]
- Predicted collisions: [which other facets/types]

Ensemble summary:
- Type distribution by triad
- Strongest predicted Collision Pairs (based on type opposition)
- Any gaps or imbalances, with justification
