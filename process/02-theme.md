# 02 THEME — Value Conflict Extraction

## Purpose

Extract the thematic core of the narrative: a Value Conflict between two genuinely positive human values that cannot both be fully realized. This is the most important conceptual phase — everything downstream derives from it.

**NO CHARACTERS in this phase.** Theme is pure argumentation. Characters come later.

---

## Step 1a: Identify Tensions

Read `wiki/projects/[PROJECT]/impulse.md` and `wiki/projects/[PROJECT]/research/`.

Identify recurring tensions, dilemmas, and value conflicts across the material. Look for:
- Where do people in this world face impossible choices?
- What values collide in the Core Finds?
- What is the argument this world is having with itself?

List every tension you find. Don't filter yet.

## Step 1b: Propose Value Conflict Candidates

From the tensions, distill 3–4 **Value Conflict candidates**. Format: `Value A vs. Value B`.

### Machine Test — Every Candidate Must Pass ALL Checks

| Test | Criterion | Pass/Fail |
|------|-----------|-----------|
| **Criterion 1: Both Positive** | BOTH values are genuinely positive. Both are things a reasonable, ethical person would defend. | If one side is obviously wrong → FAIL |
| **Criterion 2: Both Destructive** | BOTH values become destructive when taken to their extreme. Absolute loyalty destroys autonomy. Absolute freedom destroys community. | If one side has no dark extreme → FAIL |
| **Criterion 3: Genuine Tension** | Pursuing one value ACTIVELY undermines the other. Not just different priorities — active opposition. | If both can coexist peacefully → FAIL |
| **Criterion 4: Undecidable** | Neither value can be dismissed. Reasonable, intelligent people genuinely disagree about which matters more. | If one side is obviously superior → FAIL |
| **Anti-Value Check** | Neither side is an anti-value. Hatred, greed, corruption, cruelty — these are not values. They are the Dark Side of values. | If either side is an anti-value → FAIL |
| **Prohibition Test** | Not "freedom vs. oppression" (too obvious — oppression is an anti-value). Not "individual vs. collective" (too abstract — not actable). | If too obvious or too abstract → FAIL |

**Examples that PASS:**
- Loyalty vs. Justice (protecting your own vs. doing what's right)
- Truth vs. Mercy (honesty that destroys vs. lies that protect)
- Safety vs. Freedom (protection that imprisons vs. liberty that endangers)
- Tradition vs. Progress (roots that anchor vs. change that liberates)

**Examples that FAIL:**
- Good vs. Evil (not a conflict — one side is wrong)
- Love vs. Hate (hate is an anti-value)
- Freedom vs. Oppression (oppression is an anti-value)
- Individual vs. Society (too abstract to act)

---

## Step 2: Develop the Chosen Value Conflict

Select the strongest candidate. Develop FULL ambivalence argumentation for BOTH values.

### For Value A:
- **Value A saves:** Concrete scenarios where this value protects, heals, enables. Grounded in research material.
- **Value A destroys:** Concrete scenarios where this value — taken to its extreme — causes harm, oppression, suffering. Grounded in research material.
- **Value A's seduction:** Why do people cling to this value even when it hurts them? What makes it feel essential?

### For Value B:
- **Value B saves:** Concrete scenarios where this value protects, heals, enables. Grounded in research material.
- **Value B destroys:** Concrete scenarios where this value — taken to its extreme — causes harm, oppression, suffering. Grounded in research material.
- **Value B's seduction:** Why do people cling to this value even when it hurts them? What makes it feel essential?

### The Collision:
- Where in the research material do Value A and Value B collide head-on?
- Who loses when A wins? Who loses when B wins?
- What is the impossible choice that characters will face?

---

## Step 3: Derive Facets

From the fully argued Value Conflict, derive **4–5 facets** (MAXIMUM 5). Each facet is an active position within the Value Conflict — a way of living the argument.

### Facet Types:
- **Thesis facets** — Embody Value A. Actively pursue it, suffer its consequences.
- **Antithesis facets** — Embody Value B. Actively pursue it, suffer its consequences.
- **Gray zone facets** — Between or beyond both values. Caught, switching, or transcending.

### Facet Independence Test
Each facet must take a **distinct, active position** in the Value Conflict. Not a rhetorical variation of another facet.

Test: If you removed this facet, would the argument lose a voice? If yes → independent. If no → merge or replace.

### Per Facet, Document:
- Position statement: What does this facet believe about the Value Conflict? (1-2 sentences)
- Active behavior: What does someone holding this position DO? (concrete actions)
- Where they're right: When is this position genuinely the best answer?
- Where they're wrong: When does this position cause harm?
- Breaking point: What would force this position to crack?

## Step 3b: Facet Collisions

For every pair of facets that will collide in the narrative, work out **three-level ambivalence**:

| Level | Question |
|-------|----------|
| **Where A is right** | In this specific collision, what makes Facet A's position genuinely valid? |
| **Where B is right** | In this specific collision, what makes Facet B's position genuinely valid? |
| **Where neither can win** | What is the irresolvable core — the place where both are right and both cause damage? |

Not every pair needs to collide. But every facet must collide with at least one other.

---

## Step 4: Formulate Preliminary Message

The **Message** is the author's position in the Value Conflict. Stated simply and directly.

Format: "In the conflict between [Value A] and [Value B], I believe that [position]."

The message is NOT:
- A moral lesson ("Crime doesn't pay")
- A platitude ("Love conquers all")
- Neutral ("Both sides have a point")

The message IS:
- A specific, arguable position
- Embodied through the protagonist's arc outcome
- Visible through consequences, never stated by characters

---

## Quality Gate

Before completing theme, verify:

- [ ] Value Conflict traceable to impulse AND research material (not invented in abstract)
- [ ] Both values pass the Machine Test (all 6 checks)
- [ ] Full ambivalence argumentation for BOTH values (saves + destroys + seduction)
- [ ] 4–5 independent facets (each passes Facet Independence Test)
- [ ] Facet collisions with three-level ambivalence for all collision pairs
- [ ] Message formulated — specific, arguable, not neutral
- [ ] NO CHARACTERS anywhere in this document — purely argumentative
- [ ] Grounded in research material — not abstract philosophy
